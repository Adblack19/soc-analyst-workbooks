# SOC Analyst Workbooks

This repository contains structured investigation workbooks for common
Security Operations Center (SOC) alerts and events.

Each workbook documents:
- Alert purpose and trigger logic
- Step-by-step investigation methodology
- Log analysis and correlation techniques
- False positive vs true positive decision points
- Response and escalation guidance

## Who This Is For
- Entry-level SOC Analysts
- Blue team learners
- TryHackMe / home lab practitioners

## Structure
- Authentication alerts
- Endpoint detection alerts
- Network-based alerts
- Cloud security alerts

These workbooks are based on hands-on labs, threat research,
and industry-standard SOC practices.
